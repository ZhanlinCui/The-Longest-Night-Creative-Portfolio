# The Longest Night — web app

React + Vite + Tailwind. Source for the interactive archive.

- **Run:** `npm install` then `npm run dev`
- **Build:** `npm run build`
- **Specs & design docs:** `../docs/`
- **Stitch reference exports:** `../reference/stitch/`
- **Replace video/image:** add files under `public/media/` and set paths in `src/config/media.ts`
